# 🎮 Vibe Code Guardian for Zed

Game-like checkpoint system for AI-assisted coding sessions in Zed.

## Features

- **💾 Quick Save** - Create checkpoints with one keystroke
- **⏰ Auto Save** - Automatic periodic checkpoints
- **🤖 AI Detection** - Auto-checkpoint when AI makes changes
- **⏪ Rollback** - Restore code to any checkpoint instantly
- **📊 Diff View** - See what changed since any checkpoint
- **🎮 Sessions** - Organize checkpoints by coding session

## Slash Commands

- `/vibe-save` - Quick save current state
- `/vibe-checkpoint` - Create named checkpoint
- `/vibe-rollback` - Rollback to checkpoint
- `/vibe-timeline` - Show all checkpoints
- `/vibe-diff` - View changes since last checkpoint

## Installation

Install from the Zed extension marketplace or clone this repository and use "Install Dev Extension".
